# Amazon Homepage Clone

A fully responsive frontend clone of the Amazon Homepage built to practice advanced HTML layout structures and custom CSS styling.

## 🚀 Features

- **Header Component:** Clean navbar featuring the Amazon logo, country selector, search bar, and utility links.
- **Hero Section:** Large sliding background banner that matches the official look.
- **Product Grid Layout:** Multicolumn dynamic grids featuring custom boxes for different shopping categories.

## 🛠️ Tech Stack Used

- **HTML5:** Used for semantic structuring and elements.
- **CSS3:** Leveraged custom properties and grid structures
